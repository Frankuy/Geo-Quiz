# Geo-Quiz
Web-based game guessing what country is on the highlighted map. [Play it!]( https://frankuy.github.io/geo-quiz)

## How to build
```
npm run build
```

## How to start
```
npm start
```

## Technology used
* HTML
* CSS
* JavaScript
* [D3.js](https://d3js.org/)
* [webpack](https://webpack.js.org/)
